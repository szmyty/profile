GITHUB-FLAVORED MARKDOWN HTML SUPPORT SUMMARY:
================================================================================

FULLY SUPPORTED ELEMENTS:
- <div> with align attribute
- <p> with align attribute  
- <img> with src, alt, width, height attributes
- <br/> and <br>
- <table>, <tr>, <td>, <th>
- <details> and <summary> (collapsible sections)
- <kbd>, <sub>, <sup>
- HTML comments <!-- -->
- <a> with href attribute

LIMITED/DEPRECATED BUT STILL RENDERED:
- align attribute (deprecated in HTML5 but GitHub supports it)
- valign attribute (deprecated in HTML5 but GitHub renders it)
- width attribute on table cells (deprecated but functional)

NOT SUPPORTED / STRIPPED BY GITHUB:
- <style> blocks
- style="" inline attributes  
- <script> tags
- <iframe> tags
- <form> elements
- class="" attributes (ignored)
- id="" attributes (may be stripped)
- most CSS properties
- <center> tags (deprecated, use <div align="center"> instead)
